The Problem

Why EduLMS?

❌

Misplacement Errors

Students self-enrolling in wrong grade level or subject rooms.

📋

No Structure

Generic LMS platforms don't enforce the K–12 subject/grade structure.

🔓

Uncontrolled Access

Students accessing rooms they don't belong to, exposing other grade's content.

The Solution

One Subject. One Grade. One Room.

Each virtual Room is locked to exactly one subject and one grade level.
Teachers assign students — students cannot join rooms independently.

Security

Built-in Security Layers

🔐

RBAC Decorators

Every view is role-protected. Unauthorized access returns 403 Forbidden.

📧

OTP Verification

6-digit code, 10-minute expiry, 5-attempt lockout on registration.

🔗

Setup Tokens

64-byte URL-safe tokens. Single-use. 72-hour expiry for password setup.

👁️

Tab Detection

Students switching tabs during quizzes — logged, warned, reported to teacher.

🐛

Error Detection

All unhandled exceptions auto-captured with traceback, user, URL, timestamp.

🛡️

CSRF Protection

Django CSRF middleware on all forms. ngrok/production URLs whitelisted.
