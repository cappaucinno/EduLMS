Changelog

📅 All versions documented 🔄 Latest: v10.0

v10.0 Password Setup via Email Link + Profile Photos 2025
  • Removed password fields from registration form — students set their own password via secure email link after approval
  • Password setup link (72-hour expiry) sent on teacher/student approval instead of generated default password
  • Profile photo upload for both teachers and students with live preview
  • Avatar shown in topbar, links directly to profile page
  • Teacher profile page at /accounts/teacher-profile/
v9.0 Password Recovery + Student Profile 2025
  • Forgot password flow with secure reset link (1-hour expiry)
  • Student profile page with stats (rooms, submissions, quiz attempts, average score)
  • Change password tab with live strength meter
  • Superuser redirects to custom admin portal instead of Django /admin/
v8.0 Email System Fixes + Announcement Popup 2025
  • Fixed announcement email (was referencing wrong field names .body→.content)
  • Email now required in registration form; duplicate check added
  • All email sends wrapped in try/except with proper error feedback
  • Tab detection restricted to students only (not teachers)
  • Favicon added to all pages including error pages and standalone auth pages
v7.0 Announcement Popup + Tab Detection 2025
  • Bell icon with unread badge counter in topbar
  • Slide-in announcement panel with type filters (Notice/Event/Exam/Urgent/Holiday)
  • Click to open full detail modal with mark-as-read
  • Auto-popup first unread announcement on login (once per session)
  • Announcement scheduling (future date), pinning, file attachment, external link
  • Read analytics per announcement for teachers
  • Tab switch detection — full-screen warning overlay for students, logs to TabViolation table
  • Quiz page: right-click disabled, F12 and DevTools shortcuts blocked
v6.0 OTP Email Verification on Registration 2025
  • 6-digit OTP sent to student email on registration (10-minute expiry, 5-attempt limit)
  • Beautiful OTP entry page with auto-advance, paste support, countdown timer, resend cooldown
  • Teacher registration skips OTP — goes straight to pending Admin approval
  • Accounts created with unusable password until approval
v5.0 Email Notifications + Announcement System 2025
  • 13 email notification triggers (registration, approval, room assignment, grading, etc.)
  • HTML email templates with school branding
  • Announcement system with audience targeting by role and grade level
  • Teacher/Admin/Everyone email blasts on announcement post
  • Gmail SMTP support via local_settings.py override
v4.0 CSRF Fix + Teacher Approval by Admin 2025
  • Fixed CSRF 403 errors (added CSRF_TRUSTED_ORIGINS for Django 4.x)
  • Teachers now require Admin approval (previously auto-approved)
  • Students require Teacher approval
  • Register page shows role-specific approval notes dynamically
  • Admin panel enhanced with bulk approve action and pending badge
v3.0 High School Room System Redesign 2025
  • Complete redesign: Course → Room (1 subject + 1 grade level)
  • Teacher-only student assignment (no self-enrollment)
  • Grade level enforced on User model (Grade 7–12)
  • Subjects: English, Filipino, Mathematics, Science, AP, TLE, MAPEH, ESP, ICT, Research
  • Materials module (file upload + external links)
  • Room overview page for teachers
v1.0–v2.0 Initial LMS + RBAC Foundation 2025
  • Django project with Admin/Teacher/Student roles
  • Custom User model with RBAC decorators
  • Basic course, enrollment, lesson, assignment, quiz, grading system
  • Bootstrap 5 dashboard UI with sidebar navigation
  • Error detection middleware with ErrorLog database table
  • Custom 400/403/404/500 error pages
