EduLMS REST API

A JSON REST API for building mobile apps, integrations, and third-party tools on top of EduLMS. Uses JWT Bearer token authentication.

BASE URL https://your-domain.com/api/v1/
CONTENT-TYPE application/json    FORMAT JSON only

Authentication

All endpoints except POST /auth/token/ require a JWT Bearer token in the Authorization header.

# 1. Get your token
POST /api/v1/auth/token/
{"username": "your_username", "password": "your_password"}

# Response
{"access": "eyJ...", "refresh": "eyJ...", "user": {...}}

# 2. Use it in every request
Authorization: Bearer eyJ...

# 3. Refresh before it expires (1 hour)
POST /api/v1/auth/token/refresh/
{"refresh": "eyJ..."}
→ {"access": "eyJ..."}
ℹ️ Access tokens expire in 1 hour. Refresh tokens expire in 30 days and rotate on use. Store both tokens securely (e.g. iOS Keychain, Android Keystore).
⚠️ Users must be approved by a teacher/admin before they can authenticate. Unapproved users receive 403 Forbidden.

Error Codes

400Bad RequestMissing or invalid fields in request body
401UnauthorizedMissing, invalid, or expired access token
403ForbiddenAccount pending approval, or wrong role for endpoint
404Not FoundResource doesn't exist or you don't have access
